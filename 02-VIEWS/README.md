# Vistas
Aquí se encuentran los scripts para la creación de las Vistas en BD
Para los archivos sql, utilizar el formato de nombres:
>**Para archivos de creación de vistas**
>*mod##-nombre_vista.sql*

Donde _mod##_ es la abreviatura del módulo + un número secuencial que indique en que orden se deben ejecutar
Ej:
* _**gha01-pets_view.sql**_
* _**gha01-clients_view.sql**_
