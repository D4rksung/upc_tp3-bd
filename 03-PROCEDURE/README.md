# Procedimientos Almacenados
Aquí se encuentran los scripts para la creación de los Procedimientos Almacenados en BD
Para los archivos sql, utilizar el formato de nombres:
>**Para archivos de creación de vistas**
>*mod##-nombre_proc.sql*

Donde _mod##_ es la abreviatura del módulo + un número secuencial que indique en que orden se deben ejecutar
Ej:
* _**gha01-usp_GrabarReserva.sql**_
* _**gha01-usp_PlanificarHospedaje.sql**_
