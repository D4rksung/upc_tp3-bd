# Tablas
Aquí se encuentran los scripts para la creación de las Tablas en BD
Para los archivos sql, utilizar el formato de nombres:
>**Para archivos de creación de tablas**
>*mod##-create.sql*
>**Para archivos de creación de inserción**
>*mod##-insert.sql*

Donde _mod##_ es la abreviatura del módulo + un número secuencial que indique en que orden se deben ejecutar
Ej:
* _**gha01-create.sql**_
* _**gha01-insert.sql**_
